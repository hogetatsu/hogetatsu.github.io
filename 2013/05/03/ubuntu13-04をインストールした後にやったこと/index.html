ubuntu13.04(Raring Ringtail)をインストールした後にやったこと

May 3, 2013 • hogetatsu

今年も毎年恒例、ゴールデンウィークのubuntuバージョンアップです。

JapaneseTeamのリポジトリを追加

いつもお世話になっております、Japanese Team様。

sudo apt-add-repository ppa:japaneseteam/ppa
sudo apt-get update
sudo apt-get install unzip ubuntu-defaults-ja

NTPサーバをNICTに変更

初期値はubuntuのntpサーバになっていますが、昔からお世話になっている情報通信研究機構のサーバに変更します。

sudo sed -i.bak 's/NTPSERVERS=.*/NTPSERVERS="ntp.nict.jp"/' /etc/default/ntpdate

グラフィックボードのドライバ

「ソフトウェアとアップデート」を開き、「追加のドライバー」タブでプロプライエタリのドライバを選択して「変更の適用」を押します。
英語版では「Software & Updates」、「Additional Drivers」タブでした。

ホームディレクトリを英語に

ホームにある「デスクトップ」などの各種ディレクトリを英語にします。
コマンドラインで操作するときに楽なので。
やり方は簡単、nautilus(windowsで言うところのエクスプローラ)で名前を変えるだけです。
以前は

env LANGUAGE=C LC_MESSAGES=C xdg-user-dirs-gtk-update

とかで ~/.config/user-dirs.dirs を編集しましたが、13.04 では再起動時に元の日本語に戻ってしまった。
nautilusを開いて右クリックで「名前の変更」したところ、user-dirs.dir が変更されていて再起動時も英語のままでした。

アーカイバと動画

rarや7zipは使わないかもしれませんが、とりあえずインストール。必要になった時に過去の自分に少しだけイラッとしちゃうので。
同じ理由で、大抵の動画を再生できる VLC media player も。

sudo apt-get install p7zip-full rar unrar vlc

システム設定に便利なGUIツール

  • ユーザとグループ
    最初から入っている「ユーザーアカウント」を使えばGUIでユーザの追加と削除はできますが、グループの管理もしたい。
  • Tweak Tool
    フォントや時刻表示、ウィンドウに最大化と最小化ボタンを表示するなど様々な設定がGUIでできます。
sudo apt-get install gnome-tweak-tool gnome-system-tools

DNSサーバの設定

namebenchをインストール＆実行しておすすめの設定に従いました。
26.4%も速いって話なので。

Primary Server 129.250.35.250 NTT
Secondary Server 127.0.1.1 SYS-127.0.1.1
Tertiary Server 210.130.1.1 IIJ4U-11 JP
