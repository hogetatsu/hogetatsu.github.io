nlite で windows xp のインストールディスクをすっきり！！

Apr 3, 2011 • hogetatsu

いまさらながら、というか windows xp アップデートも少なくなってきた今だから？ nlite を使ってサービスパック3(SP3)やらhotfixを当てた状態のインストールディスクを作りました。

1. windows updates downloader(wud)

まず最初に wud を使ってservice pack や hotfix をまとめてダウンロードします。microsoft のダウンロードセンターでひとつずつ落とす作業を楽させてくれます。
wud を こちら から落としてインストール。
こちらのページで japanese を選択して 日本語版の sp や hotfix のリストを落として実行。

2. nlite

nlite をこちらのページ から落としてインストール。
いよいよインストールディスクイメージの作成ですが、基本的に こちら が詳しいです。基本的に次へ次へを押して進めますが、以下2点を注意しました。

  • ホットフィックスを選択しあとは必ず日時でソート
  • コンポーネントの削除でキーボードをすべて削除

3. windows インストール時に txtsetup.sif でエラー

インストールディスク読み込み時に
INI file 386txtsetup.sif is invalid
と出ました。txtsetup.sif をテキストエディタで開いて、エラー行の少し上を見るとキーボードの設定であることがわかりました。エラー行のあたりを眺めてるとダブルコーテーションついてる行とそうでない行があることに気づき、あぁなるほどな、と。でもisoファイルを直接編集する手段を知らないので、nliteのコンポーネント設定時にキーボードをすべて削除してみました。すると txtsetup.sif のエラーはなくなりましたが、windows インストール時に kbdbe.dll が見つからないと言われたので、エスケープキー を押しっぱなしにして逃げました。
