ubuntu 12.04(Precise Pangolin)のインストール直後にやったこと。

Jun 10, 2012 • hogetatsu

そろそろ落ち着いたのかなぁと思いインストールしました。

フォントとフォントサイズの変更

2つか、3つ前のバージョンだと簡単にフォントを変えることができましたが、前バージョンくらいから gnome-tweak-tool をインストールしてフォントを変えています。
インストールしたらunity(windowsキーというかスーパーキーを押して出てくるアレ)で「高度な設定」を起動します。

今回いろいろなフォントを探して見ましたが yukarrya が良さげです。ただ開発者が不明で勝手に配布している状態なようなので私は遠慮することにしました。
なのでデフォルトフォントを Migu 1C、ドキュメント/モノスペースフォントを MigMix 1M にしました。プログラマの人たちには Inconsolata が人気ようのです。

Nautilusの設定

windowsで言うところのエクスプローラです。
元々は「詳細表示」を好んでいましたが、詳細が必要な場合の多くはターミナルを使っているので「アイコン表示」にしています。
設定画面の「アイテム」タブで「見出し」を「サイズ」、「更新日時」、「場所」の順に設定します。表示を拡大(ctrl+マウスホイールをくるくる)していくと、設定した順に項目が表示されます。たまにあると便利です。

あとは日付の書式を「YYYY-MM-DD hh:dd:mm」に変えてシンプルに。

trash-cli

間違って rm しちゃった。。なんて悲劇を回避するために使っています。
alias rm=”trash-put” なんてやっておけば rm してもゴミ箱に行くので安心です。

IME

日本語imeと言えば変換能力ではatokが一番でしょう。個人用途ではちょっと高いなぁと思うのでmozcを使っています。
mozc-server mozc-utils-gui ibus-mozc をインストール。
パネル(?画面上部の時計がある並び)のキーボードの絵をクリックして設定画面。
「インプットメソッド」タブで「mozc」を追加して「anthy」を削除しました。
なおテキストエリアを選択している状態では設定画面が開かないようです。

アイコンテーマ

faenza がオススメです。
下記の通りインストールして、先ほどの「高度な設定」でfaenzaを選びます。

sudo add-apt-repository ppa:tiheum/equinox
sudo apt-get update
sudo apt-get install faience-icon-theme

guake-terminal

パッケージ名は「guake」。お気に入りのターミナルです。F12を押すと上から画面が落ちてきます。
「フォーカスを失ったら隠す」設定にするとなお良いと思います。
画面右上の歯車(シャットダウンがあるところ)に「自動起動するアプリケーション」があるのでguake terminalを登録します。

ubuntu tweak

孫の手的なアプリです。 http://ubuntu-tweak.com/からダウンロードしてインストールします。debパッケージを配布しているのでクリックするだけで簡単にインストールできます。
ホームディレクトリの各ディレクトリを変更できたりします(デスクトップ => desktop)。英語名にするとコマンド操作の時に楽ですよね。

 拡大鏡

拡大鏡と言うと少し違うのかもしれませんが。。
CompizConfig設定マネージャ(パッケージ名は compizconfig-settings-manager )をインストールします。
「Enhanced Zoom Desktop」で設定します。
マウスの zoom in を「Button4」、zoom out を「Button5」に設定しました。 これでスーパーキー(windowsキー)を押しながらマウスホイールをくるくるすると、画面が拡大/縮小されます。 画面から1~2m離れて作業する私には必須の機能です。 なおButton4とか5の番号は使っているマウスによって変わるかもしれません。

Launcherを自動的に隠す

デフォルトだと画面左端にあるアレはランチャーと呼ぶらしいです。前のバージョンだと勝手に隠れてくれた気がするのですが、今回から隠れてくれません。
デスクトップを右クリックして「背景の変更」をクリック。次に「挙動」タブで「〜自動的に隠す」のチェックボックスをオンにします。
マウスを左に持って行ってもランチャー出てこないってときは「表示感度」を高くしてあげればok。

medibuntuパッケージ
フリーではなくubuntuに含まれていないソフトウェアたちをインストールします。wmv(windows media video)のデコーダーなどです。vlc入れればそれでいい気もしますが。。
http://medibuntu.org/repository.php/ に書いてあるとおり実行します。

sudo -E wget --output-document=/etc/apt/sources.list.d/medibuntu.list http://www.medibuntu.org/sources.list.d/$(lsb_release -cs).list && sudo apt-get --quiet update && sudo apt-get --yes --quiet --allow-unauthenticated install medibuntu-keyring && sudo apt-get --quiet update
sudo apt-get install app-install-data-medibuntu apport-hooks-medibuntu
