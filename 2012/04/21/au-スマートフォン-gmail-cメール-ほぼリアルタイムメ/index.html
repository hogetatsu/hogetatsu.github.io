au + スマートフォン + gmail + cメール == ほぼリアルタイムメール

Apr 21, 2012 • hogetatsu

wimaxで通信費を節約、メールはgmailを使って絵文字も使える。とまぁ以前にもこのブログに書いきましたが、なんだかんだでキャリア提供のメールのリアルタイムなプッシュ配信は最強です。限りなくそれに近づく方法を思いついてしまったのです。

手順

  1. gmailの設定で 〜@ezweb.ne.jp へ転送する
  2. 〜@ezweb.ne.jp の設定で Cメールへ転送(Cメールアドレスは下記)

 

補足

  • gmail以外でもメール転送の機能があれば良い
  • 〜@ezweb.ne.jpの転送設定、キャリア純正のメールアプリから
  • gmailから直接Cメールへ転送することはできない
  • 3G回線を切っていてもCメールは届くので、ほぼリアルタムにメールを見れる
  • 返信はgmailなので、Cメールはメールチェッカー程度に利用
  • 手順1だけでもメールが届いたことは知らせしてくれるが、中身の確認はできない
  • Cメール用のEメールアドレスは下記の通りです。(ソースはこちら)
  • 北海道：電話番号ハイフン無し@dct.dion.ne.jp
  • 東北：電話番号ハイフン無し@tct.dion.ne.jp
  • 北陸：電話番号ハイフン無し@hct.dion.ne.jp
  • 関西：電話番号ハイフン無し@kct.dion.ne.jp
  • 中国：電話番号ハイフン無し@cct.dion.ne.jp
  • 四国：電話番号ハイフン無し@sct.dion.ne.jp
  • 九州：電話番号ハイフン無し@qct.dion.ne.jp
  • 沖縄：電話番号ハイフン無し@oct.dion.ne.jp
  • 首都圏中部：電話番号ハイフン無し@cmail.ido.ne.jp

 

思いついた経緯

話は10年以上前(?)にさかのぼります。かつてはauはidoといいました。国際電話で有名だったkdd、phsで有名だったddiポケット、それに携帯電話のidoがひとつになってkddiとなり、auという名称でサービスを展開し始めました。kdd + ddi + ido = kddi、あれ？idoはどこに行った？などと思っているとcmでも同じ事を言っていました。

その頃だった思うのですが、auでメールと言えばcメールとeメールでした。電話番号だけで送ることができるcメールの利点を信じる人がまだまだいたのです。一部ではcメールは受信が無料だから、と。メール1往復の料金合計で言えばeメールの方が安いことを知らない人たちと言えばそうですが、「相手の受信料が無料な方が良い」というのもまた日本人の美徳かもしれません。単純に手紙文化の中に生きていただけかもしれませんが。

そんな昔、cメールにもスパムメールが届いたものです。1通3円のコストをスパム業者は抱えていたのか？それが違うんです。パソコンからcメールへ送信できたんです。そんな状況でキャリア側が規制をかけ、キャリアメール(〜@ezweb.ne.jp)からのみcメールへ送信できるようにしたのです。

なんてことを思い出したのが、事の始まりでした。
