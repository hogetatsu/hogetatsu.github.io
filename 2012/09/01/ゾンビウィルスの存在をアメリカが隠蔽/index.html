ゾンビウィルスの存在をアメリカが隠蔽

Sep 1, 2012 • hogetatsu

皆で飲みながら見たTV「ウソかホントかわからない やりすぎ都市伝説スペシャル」その1。
目次はこちら。

カンニング竹山が披露した話

  1. 人間を食べる事件(通称ゾンビ事件?)が発生
  2. 事前にCDC(細菌などの研究機関)がゾンビウィルスのアウトブレイクを示唆していた
  3. ゾンビウィルスの必要性は軍事
  4. アメリカはウィルスを隠蔽

私の聞いた話

  1. CDCが「ゾンビが襲ってきても生き抜く」と題した防災アドバイスを発表
  2. 日本でも行なっている乾パンや水の備蓄を促すような普通の防災情報だった
  3. 人間を食べる猟奇的な事件が発生
  4. 「人間を食べる==ゾンビ」と関連付けて陰謀論の出来上がり
  5. 後にCDCはゾンビウィルスを公式に否定する事態に至った

私見

企業のメールマガジンなどで用いる手法ですが、時事のポップカルチャーの話題から切り出し本題を伝えることが良くあります。「ゾンビ」なんて突飛なタイトルを付けたのは映画「バイオハザード」などの話題から入り、ゾンビより怖いハリケーンに備えましょう、ゾンビに備えるくらいの気持ちで災害に備えましょう、とキャッチコピーではないかと想像します。「人間を食べる=ゾンビ」なんて関連付けるより真実味がありませんか？食人というヘンテコリンで信じがたい行為は昨日今日に始まったことではないのです。

アメリカという国は陰謀論のメッカで、映画「バイオハザード」が公開されるとゾンビウィルスの存在を信じる人が増えたと聞きます。あの映画は10年くらい前のパソコンゲームに始まったもので、科学検証のドキュメンタリーでもないのに、その映画を見てゾンビウィルスの存在を信じる思考回路は理解に苦しみます。

他にも科学番組で人魚の特集をやれば研究機関に問い合わせ殺到で、人魚否定するような事態になるお国です。

陰謀論はファンタジーな世界を楽しむことに似ているかもしれません。本当にゴブリンやスライムが襲ってくるとは思わないでしょ。

一番重要なことは発端となった猟奇事件の被害者は亡くなっていることです。弔意を持つべきであり、おおっぴらに面白おかしく夜9時に放送するようなものではありません。
